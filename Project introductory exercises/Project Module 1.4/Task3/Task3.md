# A hospital emergency service 

Service points: 

1. Triage nurse
2. Registration desk
3. Doctor examination
4. Discharge desk

Each patient goes through all the service points in the order listed above.

# Airport security check

Service points:

1. ID check
2. Security Scanner line 1
3. Security Scanner line 2
4. Secondary Inspection (optional)

Each passenger goes through the ID check, then either Security Scanner line 1 or line 2, and if flagged for additional screening, goes through the Secondary Inspection.
